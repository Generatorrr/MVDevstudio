Youtube video presenter

Приложение, которое запрашивает данные с Youtube, при помощи Youtube API.
Затем выводит результаты в виде блоков с подрробным описанием и скриншотом видео.

Calendar Application

Приложение-календарь показывает напоминает о событиях, которые должны произойти. Приложение запрашивает данные на удалённый сервер, созданный с помощью Mongo DB.
Затем выводит полученные данные в требуемые дни. События различные по типу. Справа - блок с подробной информацией.
Использованные технологии: HTML 5, CSS, Java Script, React JS, Redux JS, ReactRedux, Mongo DB, Web Pack.

Chat application with Google authorization

Приложение-чат, созданное с помощью Angular JS, показывает сообщения, введённые пользователями, авторизовавшимися с помощью аккаунта Google.
Использована технология Firebase для хранения данных. В сообщениях показано имя Google-аккаунта и аватар, если таковой имеется.
У авторизованного пользователя сообщения имеют отличный, от других пользователей, цвет сообщений.
Использованные технологии: HTML 5, CSS, Java Script, Angular JS 1.5, Firebase, AngularFire, Gulp.
